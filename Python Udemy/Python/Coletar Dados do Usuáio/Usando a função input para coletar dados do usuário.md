A tag Input cria uma pergunta ao usuário onde ele pode digitar algo e assim coletar dados.
Adicionando uma variável a função input, todos os dados que o usuário colocar dentro dessa função se tornara a variável em si. **Condição é uma pergunta**
Exemplo:

`nome = input('Qual o seu nome? ')
`print(f'O seu nome é {nome}')


`numero_1 = input('Digite o primeiro número: ')
`numero_2 = input('Digite o segundo número: ')
  
`int_numero_1 = int(numero_1)
`int_numero_2 = int(numero_2)

`print(f'A soma dos números é: {int_numero_1 + int_numero_2}')
